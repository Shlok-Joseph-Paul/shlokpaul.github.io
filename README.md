# Shlok Paul Personal Website

This is a static GitHub Pages-ready personal academic/research website.

## How to publish on GitHub Pages

1. Create a GitHub account if you do not have one.
2. Create a new repository named `yourusername.github.io`.
3. Upload all files from this folder into the repository.
4. Go to Settings → Pages.
5. Under “Build and deployment,” choose “Deploy from a branch.”
6. Choose the `main` branch and `/root` folder.
7. Save.
8. Your site should appear at `https://yourusername.github.io`.

## What to edit

- Edit `index.html` to change text, publication details, links, and contact info.
- Edit `style.css` to change visual design.
- Put your CV PDF in `assets/` and name it `Shlok_Paul_CV.pdf`.
- Replace the SP circle with a real profile photo if desired.
